## Known Issues & Mitigations

### Issue 1: Large Token Usage
**Impact**: High token consumption for large projects
**Mitigation**: Progressive disclosure, load references on demand
**Status**: Acceptable - trade-off for comprehensive guidance

### Issue 2: Skill Dependencies
**Impact**: Requires multiple nested skills
**Mitigation**: Auto-update checks, graceful degradation
**Status**: Managed - startup verification

### Issue 3: Phase 0 Complexity
**Impact**: May overwhelm users with radical AI-native ideas
**Mitigation**: Two-dimensional options (conservative vs radical)
**Status**: Designed - user choice preserved
