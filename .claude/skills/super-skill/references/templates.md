# Super-Skill Document Templates

This directory contains templates for commonly used documents in the 8-phase development workflow.

## Template Quick Reference

| Phase | Template File | Purpose |
|-------|---------------|---------|
| Phase 0 | `KNOWLEDGE_BASE.md` | Domain knowledge and technical decisions |
| Phase 1 | `REQUIREMENTS.md` | Requirements specification |
| Phase 2 | `ARCHITECTURE.md` | System architecture and design |
| Phase 2 | `API_DESIGN.md` | API specifications (if applicable) |
| Phase 2 | `DATABASE_SCHEMA.md` | Database design (if applicable) |
| Phase 3 | `TECHNICAL_IMPLEMENTATION.md` | Implementation details |
| Phase 3 | `DEVELOPMENT_PLAN.md` | Development schedule and WBS |
| Phase 3 | `TEST_PLAN.md` | Testing strategy and coverage |
| Phase 3 | `DEPLOYMENT_PLAN.md` | Deployment procedures |
| Phase 4 | `PROGRESS.md` | Daily progress tracking |
| Phase 6 | `ITERATION_LOG.md` | Iteration improvements tracking |
| Phase 8 | `PROJECT_SUMMARY.md` | Final project report |

## Template Usage

Each template includes:
- Required sections with placeholders
- Example content structure
- Formatting guidelines
- Completion criteria

Copy and customize templates for your specific project needs.
